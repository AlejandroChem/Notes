# 按机身右下底部的开关不能使康佳电视开机

## 问题
* 反复按机身底部的开关（-/o)，不能开机
  * 直线(-)是接通电源的意思，就是开机
  * 圆圈(o)是断开电源的意思，就是关机

## 原因
* 康佳电视正在“睡眠”模式

* 电视机进入“睡眠“模式的途径

  | 序号 |  途径 |
  | :--: | :--: |
  | 1 | 电视机长时间没有操作，自动进入”睡眠“模式 |
  | 2 | 按遥控器上的左上角的“睡眠”按钮 |
  | 3 | 按电视机左下侧面的最下面（第7个）“睡眠”按键 |

## 解决方法
* 再次唤醒电视机

  | 序号 |  途径 |
  | :--: | :--: |
  | 1 | 按遥控器上的左上角的“睡眠”按钮 |
  | 2 | 按电视机左下侧面的最下面（第7个）“睡眠”按键 |
