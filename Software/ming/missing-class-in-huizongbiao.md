# 班级在汇总表（按班级）中消失了

## 背景
* 明日系统(ming800)定制版

## 问题
* 某个班级，在“汇总表（按班级）”中消失了
* 但是这个班级，在“当前专业与班级”的列表中显示正常
* 点击班级名称进入查看，发现“课程”缺失，没有记录

## 解决
* 设置班级的”课程“即可
  * 在查看页面，点击左上角的“修改”
  * 在“课程”栏，勾选课程记录
* 只能使用IE浏览器
* Win10自带的Edge，勾选框不能显示，所以也不能修改
* Win10运行IE的方法:
  * 按"Win"(左下角第2个窗口)键 + "s"键
  * 在弹出的搜索框，输入"ie"
  * 提示"Internet Explorer"，点击即可运行IE
